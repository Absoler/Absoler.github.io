组合计数积累

Posted on 2020-05-27 | In 数学

1

https://codeforces.com/problemset/problem/1358/C

最开始想到了棋盘走法问题，那个答案是C(n+m, m)，于是便想着把这道题区域中过对角线的部分中可能sum重复的走法剪掉，后来发现不可取，

解法很巧妙，先选中sum最小的一种（沿着顶边和右边），接下来如果想扩大，只能把顶点向左下“内缩”，这样sum可以加一，同时我们知道最大的sum，在最小最大的这个区间中，每个sum均可以通过从最小sum缩若干次达到，所以sum不同的方案数就是这个区间的范围。“In order for us to come from the minimum to the maximum way, we must bend the way exactly 1 time per each cell of table (except for the cells of the minimum way). That is, the number of changes equals the number of cells not belonging to the minimum way”这是算最大最小差值的方法。
